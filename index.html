Cursos grátis

Curso de HTML5 e CSS3

A HTML é uma linguagem de marcação utilizada para criar páginas na web. Atualmente na versão 5, a HTML5 trouxe diversas melhorias e novas funcionalidades para os desenvolvedores. Já o CSS é uma linguagem de estilo utilizada para definir a aparência das páginas web. O CSS3 é a versão mais recente, que introduziu novos recursos e propriedades para tornar o design mais flexível e responsivo. Juntos, HTML5 e CSS3 são essenciais para criar sites modernos e atraentes.

Curso de JavaScript

JS é uma linguagem de script criada para aumentar a interatividade entre os usuários e os elementos de uma página web. Os códigos em JS sõ executados dentro do navegador e dão acesso aos elementos de HTML por meio do DOM.
